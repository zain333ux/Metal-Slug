## About the Project

The game is based on a run-and-gun style where the player fights different enemies, uses multiple weapons, drives vehicles, survives different biomes, and battles bosses. The main goal of this project was not only to create a playable game but also to build it with a clean object-oriented structure.

The project includes different gameplay systems such as:

- Playable soldiers
- Enemy soldiers and special enemies
- Vehicles
- Weapons and projectiles
- Boss fights
- Collectibles
- Sound effects and background music
- Game states such as menu and play state
- Scrolling level system
- Biome-based gameplay
- Transformation states such as zombie and mummy effects

## Demo Controls

- Menu: Up/Down, Enter
- Move: A/D or Arrow keys
- Jump: W, Up, or Space
- Shoot pistol: J or Left Ctrl
- Knife melee: K
- Grenade: G
- Rocket: H
- Switch character: Z
- Toggle Developer Mode: press F1 three times within two seconds
- Return to menu: Escape

## Implemented Demo Path

Start Survival, switch characters, collect the crate/POW/food pickups, kill enemy batches with pistol/melee/grenades/rockets, reach the right boundary to clear levels, then continue into the boss level. Campaign mode starts a generated profile level and awards campaign-clear score when cleared.
