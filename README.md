# OOP Project

## Demo Controls

- Menu: Up/Down, Enter
- Move: A/D or Arrow keys
- Jump: W, Up, or Space
- Shoot pistol: J or Left Ctrl
- Knife melee: K
- Grenade: G
- Rocket: H
- Switch character: Z
- Toggle Developer Mode: press F1 three times within two seconds
- Return to menu: Escape

## Implemented Demo Path

Start Survival, switch characters, collect the crate/POW/food pickups, kill enemy batches with pistol/melee/grenades/rockets, reach the right boundary to clear levels, then continue into the boss level. Campaign mode starts a generated profile level and awards campaign-clear score when cleared.
